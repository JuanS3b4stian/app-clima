Proyecto de Clase: App del Clima con OpenWeatherMap

Construiremos desde cero una aplicación web que consulta la temperatura real de cualquier ciudad
del mundo usando la API pública de OpenWeatherMap. El usuario escribe la ciudad, selecciona el
país, y la app muestra la temperatura actual, máxima y mínima en tiempo real.

---

Estructura del proyecto:
📁 clima-app/
├── index.html ← Estructura y formulario
├── css/
│ ├── styles.css ← Estilos propios + Spinner
│ └── tailwind.min.css ← Framework CSS (provisto por el docente)
└── js/
└── app_clima.js ← Toda la lógica JavaScript
